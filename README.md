# C1 驾照 · 科目一（智能刷题 + 模拟考试 · PWA）

一个零依赖、可离线、可安装的科目一考试 PWA 网页。支持 **SM-2 间隔重复智能刷题**、**仿真模拟考试**、**错题重练**三种模式，CDN 图片自动缓存本地，手机安装后可完全替代驾考 App 做日常练习。

## 🚀 在线使用

访问 **[https://agentmystia.github.io/driving-test-subject1/](https://agentmystia.github.io/driving-test-subject1/)**（GitHub Pages 自动部署）

**安装到手机桌面**（推荐，体验等同原生 App）：
- **iOS Safari**：打开网址 → 分享 → 「添加到主屏幕」
- **Android Chrome**：打开网址 → 菜单 → 「添加到主屏幕」/「安装应用」
- **桌面 Chrome/Edge**：地址栏右侧安装图标 → 安装

安装后全屏启动、无浏览器地址栏，离线可用（首次需联网加载）。

## 三种模式

### 📚 刷题练习（智能精选）
基于 **SM-2 间隔重复算法**抽题，用最少时间巩固最多知识点：
- 优先复习**今日到期**的题目（错过的题间隔到期需要巩固）
- 引入适量**新题**（控制每日新题量，避免过载）
- 答题**即时判定**：答对变绿、答错所选变红并标出正确答案
- 立即展开**正确答案 / 解析 / 法条依据**
- 不限时，节奏完全可控

### 📝 模拟考试（仿真）
对标真实上机考试：
- 判断题 **40** + 单选题 **60**，章节分层抽样
- 限时 **45 分钟**，到点自动交卷
- 答题不揭示对错，交卷后统一评定
- 分数环 + 对错统计 + 逐题解析（错题置顶、可筛选）
- 答错的题自动注入 SM-2 复习队列

### ❌ 错题重练
复习 SM-2 算法判定**今日到期**的题目：
- 答对推进间隔（间隔变长，未来少出现）
- 答错重置间隔（明天再来）
- 全部答对后首页"今日待复习"清零

## 🧠 SM-2 间隔重复算法原理

每题记录难度系数（ease）与复习间隔（interval）：

| 行为 | 算法响应 |
|------|---------|
| 答对 | interval 增长：1天 → 3天 → 7天 → 16天 → 35天…（按 ease 倍增），ease 微增 |
| 答错 | interval 重置为 1 天，ease 降低（最低 1.3） |

效果：**已掌握的题间隔越来越长、少打扰；薄弱的题频繁出现、强化记忆**。科学依据艾宾浩斯遗忘曲线，避免重复刷已会的题，专注薄弱点。

掌握判定：连续答对 ≥3 次且间隔 ≥7 天 → 标记为"已掌握"。

## 🖼️ 图片离线缓存

题目图片来自 CDN，采用 **Service Worker + Cache API** 实现本地缓存：

- **懒缓存**：看一题自动缓存一题，下次离线也能看
- **一键预下载**：首页点「📥 预下载图片库」可批量下载全部 714 张图片（约 50-100MB），完成后完全离线可用
- 预下载带进度条，可随时取消

## 📊 学习数据持久化

所有数据存储在浏览器本地（localStorage + Cache API），不上传任何服务器：
- SM-2 间隔记录（每题的 ease/interval/reps/due）
- 答题统计（累计正确率、模考历史、连续学习天数）
- 图片缓存（Cache API）

首页可「↺ 重置进度」清空学习记录重新开始。

## 本地开发与构建

```bash
python3 scripts/fetch_bank.py    # 下载/刷新原始题库（可选，data/ 已含）
python3 scripts/classify.py      # 规范化与分类（可选）
python3 scripts/build_html.py    # 生成 index.html / manifest.json / sw.js / 图标
```

依赖：Python 3.6+ + Pillow（生成图标用，`pip install Pillow`）。

**本地预览**：构建后双击 `index.html` 即可（功能正常，但 Service Worker 仅在 http/https 下生效）。
完整 PWA 体验需起本地服务：`python3 -m http.server` 后访问 `http://localhost:8000`。
